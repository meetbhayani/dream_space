# DESIGN_SYSTEM.md

# Dream Space Studio — Design System v1.0

---

# PURPOSE

This document defines the complete visual language of the website.

Every page, component, animation, and layout must follow this design system.

Never create UI outside these rules unless explicitly required.

The website should feel like a premium architectural experience, not a standard corporate template.

---

# DESIGN PRINCIPLES

The interface must always feel:

• Elegant

• Minimal

• Spacious

• Architectural

• Editorial

• Premium

• Timeless

• Calm

• Sophisticated

Avoid:

× Cheap gradients

× Bright colors

× Cartoon UI

× Generic Bootstrap layouts

× Over-designed cards

× Busy pages

× Heavy borders

# DEFINITION OF LUXURY

Luxury means restraint.

More whitespace.

Better typography.

Fewer elements.

Higher quality imagery.

Purposeful motion.

Perfect alignment.

Timeless colors.

Calm interactions.

Every element should feel intentional.

If a component does not improve the experience, remove it.

End of Design System.
